============
Installation
============


Stable release
--------------

To install Keyword Clustering with Transformers, run this command in your terminal:

.. code-block:: console

    $ pip install keyword_clustering_with_transformers

This is the preferred method to install Keyword Clustering with Transformers, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Keyword Clustering with Transformers can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/francesco-bosia/keyword_clustering_with_transformers

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/francesco-bosia/keyword_clustering_with_transformers/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/francesco-bosia/keyword_clustering_with_transformers
.. _tarball: https://github.com/francesco-bosia/keyword_clustering_with_transformers/tarball/master
